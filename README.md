# hoshi unlocker

Расширение для доступа к некоторым плеерам на сайте hoshi.tv

## Установка

1. Скачайте https://github.com/hoshi-app/UnlockExtension/archive/refs/heads/main.zip
2. Распакуйте в стабильное место, например в корень диска или в папку "Документы"
3. Откройте `chrome://extensions` в браузере
4. Включите **Режим разработчика** (переключатель в правом верхнем углу)
5. Нажмите **Загрузить распакованное расширение**
6. Выберите распакованную папку с расширением
7. Готово — иконка появится в тулбаре
